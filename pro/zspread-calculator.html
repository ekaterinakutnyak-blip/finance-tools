Денежные потоки и дисконтирование

Данные получены через открытый API Московской биржи (ISS MOEX) в реальном времени. КБД строится по модели Нельсона–Сигеля–Свенссона.
Z-spread подбирается методом бисекции как постоянная надбавка к КБД, при которой PV денежных потоков = грязной цене.
Risk-neutral PD из Z-spread обычно выше реальной (physical) PD из-за премии за кредитный риск.
